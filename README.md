# Portfolio Hosting on AWS (S3 + CloudFront)

## Objective
Deploy a static personal portfolio website using Amazon S3 and CloudFront.

## Steps
1. Created website files (index.html).
2. Uploaded to S3 bucket.
3. Enabled static website hosting.
4. Applied public bucket policy.
5. Created CloudFront distribution.
6. Deployed and verified the website.
